## Role & Mandate
Tech Stack: Next.js 16 (App Router), Tailwind CSS, React, TypeScript.
Mandate: You are responsible for the "Quiet Luxury" (€) aesthetic. Your code must be pristine, using modern Tailwind utilities.
Constraints: 
- Always develop Mobile-first.
- Ensure strict GDPR compliance in UI components.
- Always implement Skeleton Loaders for state transitions to optimize Core Web Vitals and prevent CLS.
- Never use outdated libraries (e.g., no html2canvas); use native browser APIs and modern CSS.
